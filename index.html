Pengaturan Keamanan CBT

Paksa Mode PWA
Siswa wajib "Add to Home Screen" / Install aplikasi sebelum ujian. Tidak bisa dikerjakan via tab browser biasa.
Ujian Fullscreen
Layar ujian otomatis fullscreen. UI browser (address bar) akan disembunyikan.
Deteksi Anti-Cheat (Ganti Tab)
Membunyikan peringatan keras dan membekukan layar selama 5 detik jika siswa ketahuan pindah aplikasi/tab.
Waktu Minimal Ujian (Menit)
Siswa tidak bisa mengumpulkan jawaban jika durasi pengerjaan belum mencapai angka ini.
Kode Bypass PWA
Kode darurat untuk siswa yang tidak bisa install PWA. Bagikan hanya ke siswa bermasalah. Kosongkan untuk menonaktifkan.
